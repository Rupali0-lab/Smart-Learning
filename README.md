# Smart-Learning

Project Description:

This project is a mobile application development project with the goal of creating a mobile app that assists students in their studies by effectively combining learning and productivity features. The application helps students create and manage study notes, plan their studies, set reminders, and monitor their academic performance.

App Modules:

1. User Authentication Module
This module enables the user to safely log in and manage their personal profiles.

2. Notes Management Module
In this module, users can create and manage their own study notes.

3. Study Planner & Reminder Module
This module allows users to plan their studies and set deadlines with regards to completing tasks.

4. Gamification Module 
This module offers users points, levels, and badges to motivate users to build regular habits of studying. 

5. Notification Module 
This module includes the option for the application to send alerts or reminders concerning its study plan. 

6. Analytics & Insights Module 
This module measures the level of performance and gives insights regarding the level of understanding by the user.
